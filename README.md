# Explain Yourself Site

Single-page GitHub Pages landing page for **Explain Yourself**, built with plain HTML and CSS.

## Files

- [index.html](/Users/peuphosin/ExplainYourselfGameWEBSITE/explain-yourself-site/index.html)
- [styles.css](/Users/peuphosin/ExplainYourselfGameWEBSITE/explain-yourself-site/styles.css)

## Edit the page

- Update the iOS and Android button links in [index.html](/Users/peuphosin/ExplainYourselfGameWEBSITE/explain-yourself-site/index.html).
- Update the support email in [index.html](/Users/peuphosin/ExplainYourselfGameWEBSITE/explain-yourself-site/index.html) if needed.

## Preview locally

Open [index.html](/Users/peuphosin/ExplainYourselfGameWEBSITE/explain-yourself-site/index.html) directly in a browser.

## Deploy

1. Push to `main`.
2. In GitHub Pages settings, use **GitHub Actions** as the source.
3. The workflow at [.github/workflows/deploy.yml](/Users/peuphosin/ExplainYourselfGameWEBSITE/explain-yourself-site/.github/workflows/deploy.yml) uploads the static files directly.
